# Order System

A Windows Forms application built with Visual Studio Community 2022 that allows users to order products from a database.

## Features

- **User Authentication**: Register and login to access the system
- **Product Ordering**: Browse and order products from the database
- **Database Integration**: Secure data storage and retrieval

## Requirements

- Visual Studio Community 2022
- .NET Framework
- SQL Server (or compatible database)

## Getting Started

1. Clone or download the project
2. Open the solution in Visual Studio Community 2022
3. Configure your database connection
4. Build and run the application

## Usage

1. Register a new account or login with existing credentials
2. Browse available products
3. Place orders through the application
